# JavaScript: Password Generator

An application that enables users to generate random passwords based on criteria that they’ve selected. This app runs in the browser and outputs the random passowrd through use of dynamically updated HTML and CSS powered by JavaScript code.

The password can include special characters. If you’re unfamiliar with these, see this [list of password special characters](https://www.owasp.org/index.php/Password_special_characters) from the OWASP Foundation.

## Installation
Visit the site on github pages at: https://jesse-dejong.github.io/Password-Generator/

## Usage
Site contains only one page which can be interacted with through inputting data in the available prompt fields.
